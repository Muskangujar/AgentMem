# AgentMem

AgentMem is a powerful memory management system for AI agents, featuring a Rust core and multi-language SDKs.

## Structure

- `core/`: Rust-based memory engine.
- `sdk/python/`: Python SDK with integrations for AgentID, CrewAI, and LangGraph.
- `sdk/typescript/`: TypeScript SDK and CLI.

## Development

### Core
```bash
cd core
cargo build
```

### Python SDK
```bash
cd sdk/python
pip install -e .
```
